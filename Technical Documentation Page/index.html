Introduction

Markdown is a lightweight markup language with plain-text formatting syntax. It is widely used for formatting readme files, writing messages in online forums, and creating rich text using a plain-text editor. This documentation provides a comprehensive overview of Markdown syntax and usage.

What You Should Already Know

Before using Markdown, it's beneficial to have a basic understanding of:

  • Plain-text editing: Markdown uses simple symbols to format text, making it easy to learn and use.
  • HTML basics (optional): Understanding HTML can help you understand how Markdown converts to HTML.
Markdown Basics

Markdown syntax consists of simple formatting rules to denote text elements and structure. These basic elements form the foundation of Markdown.

Headers

Headers are used to denote sections of a document. There are six levels of headers, denoted by one to six hash symbols (#). The more hash symbols, the smaller the header level:

# Header 1 ## Header 2 ### Header 3
Emphasis

Markdown allows you to emphasize text using asterisks (*) or underscores (_):

  • italic or italic
  • bold or bold
Lists

Markdown supports both ordered and unordered lists:

  • Unordered list:
    • - Item 1 - Item 2
  • Ordered list:
    • 1. First item 2. Second item
Links

Create links using [link text](URL) syntax:

[Google](https://www.google.com)
Images

Embed images using ![alt text](image URL) syntax:

![Markdown Logo](https://markdown-here.com/img/icon256.png)
Advanced Markdown

Markdown offers additional features for more complex formatting.

Code Blocks

Tables

Create tables with pipes (|) and hyphens (-):

| Header 1 | Header 2 | | -------- | -------- | | Cell 1 | Cell 2 |
Blockquotes

Use angle brackets (>) for blockquotes:

> This is a blockquote.
